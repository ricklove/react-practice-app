Practice React App

### Commands

- running app
    - `npm start`
- component testing
    - `npm test`
- debug with vscode
    - choose 'Debug CRA Tests'
    - set breakpoints
    - F5
- debug tests with vscode
    - choose 'Debug CRA Tests'
    - set breakpoints
    - F5
